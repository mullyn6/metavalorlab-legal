Términos de Servicio – MetaValorLab AutoPoster

Última actualización: 01/04/2026

1. Descripción del servicio

MetaValorLab AutoPoster es una herramienta de automatización que permite publicar contenido educativo sobre finanzas personales e inteligencia artificial en TikTok mediante la Content Posting API de TikTok. La autenticación se realiza a través de Login Kit (OAuth 2.0).

2. Uso aceptable

Este servicio está destinado exclusivamente al uso por parte del titular de la cuenta TikTok @metavalorlab. No está disponible para uso público ni para terceros.

3. Datos y privacidad

La aplicación accede únicamente a los datos necesarios para la publicación de contenido: identificador de usuario (open_id) y nombre de visualización. No almacenamos ni compartimos datos personales con terceros. Consulta nuestra Política de Privacidad para más información.

4. Autenticación TikTok

El usuario autoriza el acceso mediante OAuth 2.0 a través de Login Kit de TikTok. Los permisos solicitados son: user.info.basic, video.publish y video.upload. El usuario puede revocar el acceso en cualquier momento desde la configuración de su cuenta TikTok.

5. Publicación de contenido

El contenido publicado mediante esta aplicación es generado y revisado por el titular de la cuenta. La aplicación actúa como intermediario técnico y no es responsable del contenido publicado.

6. Modificaciones

Nos reservamos el derecho de modificar estos términos. Los cambios serán notificados mediante actualización de esta página.
